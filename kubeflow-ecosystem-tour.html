Just 5 Lines of Code

Fine-tune LLMs with the Python SDK

from kubeflow.trainer import TrainingClient
 
client = TrainingClient()
 
client.train(
  model="meta-llama/Llama-3.2-1B",
  dataset="tatsu-lab/alpaca",
  num_nodes=4
)
 
# That's it! 🚀
